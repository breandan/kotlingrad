Rational numbers that may turn to null if they get too big. For many operations, if the length of the nuumerator plus the length of the denominator exceeds a maximum size, we simply return null, and rely on our caller do something else. We currently never return null for a pure integer or for a BoundedRational that has just been constructed.

We also implement a number of irrational functions. These return a non-null result only when the result is known to be rational.
