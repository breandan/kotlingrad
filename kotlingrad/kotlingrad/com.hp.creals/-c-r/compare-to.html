Return 0 if x = y to within the indicated tolerance,-1 if x < y, and +1 if x > y. If x and y are indeedequal, it is guaranteed that 0 will be returned. Ifthey differ by less than the tolerance, anythingmay happen. The tolerance allowed isthe maximum of (abs(this)+abs(x))*(2**r) and 2**a
